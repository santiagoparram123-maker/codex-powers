Use Codex Powers to inspect this project, clarify the feature requirements, create a brief implementation plan, add tests where practical, implement the feature, and summarize verification.

## Bug Fix

```text
Use the bug investigation workflow. Reproduce the issue if possible, identify the cause, make the smallest fix, add or update a regression test, and report the result.
```

## Landing Page

```text
Use the landing page workflow. Build a conversion-focused page, keep the copy specific to the offer, check responsive behavior, and report remaining launch risks.
```

## Code Review

```text
Review my current diff. Put findings first, ordered by severity, with file and line references where possible. Include test gaps and security concerns.
```
